- 추후 설명 추가 예정

1. 로그인 / 로그아웃
2. 게시글 CRUD
3. 게시글 댓글 CRUD
4. 옷 CRUD
5. 옷 통계 산출

# 📌 실행 시 주의사항

AWS t2.small 인스턴스 (RAM 용량 2GB)를 사용하였습니다.

따라서 용량이 너무 큰 이미지를 업로드하면 에러가 발생할 수 있습니다!

이는 인스턴스의 RAM 용량 이슈로, 이미지 업로드 시 에러가 발생할 경우 용량이 작은 사진을 업로드 해주세요!
